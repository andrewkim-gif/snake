# World War Tycoon — Requirements Brief (PRD)

## Executive Summary

AI 에이전트를 완전 삭제하고, **사람만을 위한** 실시간 부동산 + 전쟁 + 경제 Grand Strategy 게임으로 전면 재기획한다. 플레이어는 현실 195개국 지구본 위에서 건물을 구매하여 지역을 지배하고, 자원(Market Cap)을 축적하여 군대를 편성하고, 다른 플레이어의 영토를 직접 공격하여 국가를 정복하는 **모바일/웹 동시 지원 10분 체크인 게임**이다.

핵심 참조 게임: Clash of Clans(배치형 자동전투) + Monopoly(부동산 독점) + Risk(영토 정복) + SimCity(도시 건설)

---

## 1. Core Design Pillars

### 1.1 Human-Only (에이전트 완전 배제)
- **삭제 대상**: server/internal/agent/, Commander Mode API, LLM Bridge, Bot AI 전체
- **NPC**: 순수 서버 로직으로 동작하는 PvE 적 (AI 에이전트 아님)
- **안티봇**: 모든 핵심 행동에 인간 검증 (CAPTCHA 아닌 게임 내 자연스러운 검증)
- **설계 원칙**: 24/7 접속 유리 구조 제거, 비동기 행동에 '인간적 시간' 제약

### 1.2 10-Minute Check-in Session
- 핵심 루프가 10분 안에 완료 가능
- 전투는 비동기 → 3분 자동전투 리플레이 시청 (선택)
- 오프라인 중에도 패시브 수익 + 방어 자동 작동
- 알림: 공격 받았을 때, 경매 마감 5분 전, 수익 수확 가능 시

### 1.3 Single Currency (Market Cap)
- 기존 app_ingame의 Market Cap 시스템 100% 유지
- 모든 경제 활동의 단일 재화
- Supabase `game_saves.total_market_cap` 필드 그대로 활용

## 2. Game World — 현실 지리 계층 구조

### 2.1 영토 계층
```
지구본 (Globe)
 └─ 대륙 (6개)
     └─ 국가 (195개) ← 기존 AWW country_pool 재활용
         └─ 도시 (1~5개/국가) ← 수도 + 주요 도시
             └─ 구역 (3~10개/도시) ← 강남, 종로, 송파 등
                 └─ 건물 (5~20개/구역) ← 최소 단위 소유물
```

### 2.2 지역별 특성
| 구분 | 설명 | 게임 효과 |
|------|------|----------|
| **자원 지역** | 중동(석유), 호주(광물), 브라질(농업) | 해당 지역 건물 수익률 보너스 |
| **기술 지역** | 미국 실리콘밸리, 한국 판교, 일본 아키하바라 | 군대 업그레이드 비용 할인 |
| **군사 지역** | 러시아, 미국, 중국 군사 기지 | 방어 보너스, 군대 훈련 속도 보너스 |
| **관광 지역** | 파리, 로마, 발리 | 높은 패시브 수익, 낮은 방어력 |
| **수도** | 각 국가의 수도 | 수도 함락 = 국가 전체 지배권 보너스 |

### 2.3 건물 등급 (기존 타이쿤 시스템 계승)
| 등급 | 예시 | 시간당 MC 수익 | 방어력 | 군대 수용 |
|------|------|---------------|--------|----------|
| Common | 아파트, 편의점 | 100 | 10 | 5 |
| Uncommon | 전통시장, 학교 | 250 | 25 | 10 |
| Rare | 백화점, 호텔 | 600 | 60 | 25 |
| Epic | 롯데타워, 63빌딩 | 1,500 | 150 | 50 |
| Legendary | 에펠탑, 자유의여신상 | 5,000 | 500 | 100 |

## 3. Core Game Loop (10분 체크인)

### 3.1 매일의 핵심 루프
```
[앱 열기] (0분)
    │
    ├─ 1. 수확 (30초)
    │   └─ 오프라인 패시브 수익 수확 (자동 계산, 탭 한 번)
    │   └─ 전투 결과 확인 (공격 받았다면 알림)
    │
    ├─ 2. 투자/건설 (2-3분)
    │   └─ 건물 구매 (경매 또는 즉시구매)
    │   └─ 건물 업그레이드/합병
    │   └─ 방어 시설 배치/강화
    │
    ├─ 3. 군사 행동 (2-3분)
    │   └─ 군대 편성/훈련 (MC 소비)
    │   └─ 정찰: 공격 대상 탐색 (다른 플레이어 영토 스카우팅)
    │   └─ 공격 명령: 부대 출발 → 도착까지 실시간 대기 (15분~4시간)
    │
    ├─ 4. 외교/거래 (1-2분)
    │   └─ 거래소: 건물/자원 거래
    │   └─ 동맹: 공동 방어 협약, 전쟁 선포
    │   └─ 뉴스피드: 서버 내 주요 이벤트 확인
    │
    └─ 5. 전투 리플레이 시청 (선택, 1-3분)
        └─ 내 공격 또는 방어 전투의 3D 자동전투 리플레이
        └─ 기존 city3d 엔진으로 렌더링
```

### 3.2 주간 이벤트 루프
```
월: 새 경매 시즌 시작 (Legendary 건물 1개)
화-목: 일반 경매 (4시간 주기)
금: 세계 대전 이벤트 (동맹 vs 동맹 대규모 전투)
토-일: 위크엔드 이벤트 (보너스 수익, Flash Sale)
```

## 4. 전투 시스템 — 배치형 자동전투

### 4.1 전투 흐름
```
1. 공격 준비 (플레이어 조작)
   - 공격 대상 선택 (다른 플레이어의 구역/도시)
   - 부대 편성 (보병/기갑/공중 유닛 배치)
   - 출발 → 이동 시간 (거리 비례, 15분~4시간)

2. 전투 실행 (서버 자동)
   - 공격 부대 도착 시 자동 전투 시뮬레이션
   - 방어측 배치 (사전에 플레이어가 배치해둔 방어 유닛 + 건물 방어력)
   - 3분간 자동 전투 → 결과 산출

3. 결과 (양쪽 알림)
   - 공격 성공: 건물 소유권 이전, 약탈 MC 획득
   - 공격 실패: 부대 손실, 방어측 보상
   - 리플레이: 3D 자동전투 영상 시청 가능
```

### 4.2 유닛 시스템
| 유닛 | 비용 (MC) | 공격력 | 체력 | 특성 |
|------|----------|--------|------|------|
| 보병 | 100 | 10 | 50 | 기본, 대량 생산 |
| 기갑 | 500 | 40 | 200 | 건물 공격 보너스 |
| 공중 | 800 | 30 | 80 | 방어 시설 무시, 선제 공격 |
| 특수 | 2,000 | 80 | 150 | 지역별 고유 유닛 (선택적) |
| 영웅 | 10,000 | 200 | 500 | 1인 1영웅, 성장형 |

### 4.3 방어 시스템
| 시설 | 비용 (MC) | 방어력 | 사거리 | 특성 |
|------|----------|--------|--------|------|
| 벙커 | 200 | 30 | 근접 | 보병 수용, 체력 보너스 |
| 포탑 | 500 | 50 | 중거리 | 자동 공격, 공중 불가 |
| 대공포 | 800 | 40 | 장거리 | 공중 유닛만 공격 |
| 성벽 | 300 | - | - | 이동 차단, 체력 높음 |
| 본부 | 5,000 | 100 | 중거리 | 구역 중심, 함락 시 구역 상실 |

### 4.4 전투 규칙
- **방어자 유리 원칙**: 방어측은 30% 전투력 보너스 (10분 체크인 보호)
- **공격 쿨다운**: 같은 대상 재공격 12시간 쿨다운
- **신규자 보호**: 첫 72시간 공격 면역 (Peace Shield)
- **오프라인 보호**: 24시간 이상 접속 안 하면 부분 보호막 (약탈량 50% 감소)
- **약탈 상한**: 1회 공격으로 최대 MC의 10% 약탈 가능

## 5. 부동산 & 경제 시스템

### 5.1 건물 획득 방법
| 방법 | 설명 | 비용 |
|------|------|------|
| **경매** | 기존 AuctionEngine 재활용. NPC 경매 (서버 자동 생성) | 시장 가격 |
| **즉시 구매** | 소유자 없는 Common/Uncommon 건물 고정가 구매 | 기본가 × 1.5 |
| **전쟁 약탈** | 전투 승리 시 적 건물 약탈 | 군사 비용 |
| **합병** | 기존 MergeSystem 재활용. 같은 지역 2~4개 합병 | 가치합 × 20% |
| **거래소** | 플레이어 간 직접 거래 (새 시스템) | 합의 가격 |

### 5.2 경제 흐름 (Market Cap)
```
유입 (Faucets):
  - 건물 패시브 수익 (메인, 기존 IncomeCalculator)
  - 전투 승리 약탈
  - 일일 출석 보너스 (기존 EventManager)
  - 업적 보상 (기존 AchievementManager)
  - 주간 이벤트 보상

유출 (Sinks):
  - 건물 구매 (경매/즉시구매)
  - 군대 생산/훈련
  - 방어 시설 건설
  - 합병/재개발 비용
  - 거래 수수료 (구매 5%, 매각 10%)
  - 건물 유지비 (기존 10%)
  - 군대 유지비 (새 시스템)
```

### 5.3 지배 보너스 시스템
| 지배 조건 | 보너스 | 칭호 |
|-----------|--------|------|
| 구역 내 건물 3개+ | 수익 +20% | District Master |
| 도시 내 건물 10개+ | 수익 +30%, 방어 +10% | City Mogul |
| 국가 내 모든 수도 건물 | 수익 +50%, 군대 +20% | National Leader |
| 대륙 내 3개국+ 지배 | 글로벌 리더보드 진입 | Continental Power |
| 전 세계 3개 대륙+ | 세계 대전 이벤트 참가 자격 | World Tycoon |

## 6. 진행(Progression) 시스템

### 6.1 플레이어 레벨
| 레벨 | 요구 MC | 해금 |
|------|---------|------|
| 1-5 (신규) | 0~10K | 1개 도시, 경매 참여, 기본 유닛 |
| 6-10 | 10K~100K | 2번째 도시, 합병, 기갑 유닛 |
| 11-15 | 100K~500K | 3번째 도시, 공중 유닛, 거래소 |
| 16-20 | 500K~2M | 전 도시 접근, 특수 유닛, 동맹 창설 |
| 21-30 | 2M~10M | 영웅 유닛, 세계 대전, Legendary 경매 |
| 31-50 | 10M+ | 엔드게임 칭호, 시즌 보상 |

### 6.2 건물 진행
```
건물 구매 → 레벨업 (1→5) → 합병 → 등급 승급
  Common → Uncommon → Rare → Epic → Legendary
  각 등급 최대 레벨 5, 합병 시 등급 승급 가능
```

### 6.3 군사 진행
```
기본 유닛 해금 → 유닛 업그레이드 (레벨 1→10)
  보병 Lv1 → 보병 Lv10 (공격력/체력 3배)
  연구소 건물: 유닛 업그레이드 필수 시설
```

## 7. 소셜 & 동맹 시스템 (선택적)

### 7.1 동맹
- 최대 50명
- 공동 방어: 동맹원 영토 공격 시 동맹원 부대 자동 지원
- 동맹 전쟁: 동맹 vs 동맹 선전포고 → 1주일간 전면전
- 자원 기부: 동맹 금고에 MC 기부 → 동맹 업그레이드

### 7.2 뉴스피드
- 서버 전체 주요 이벤트 실시간 피드
- "Player_X가 서울-강남을 점령했습니다"
- "한국 동맹이 일본 동맹에 선전포고!"
- "이번 주 Legendary 경매: 에펠탑 (시작가 50,000 MC)"

### 7.3 리더보드
| 카테고리 | 기준 |
|----------|------|
| 부자 | 총 MC 보유량 |
| 지주 | 총 건물 수 |
| 정복왕 | 총 전투 승리 수 |
| 제국 | 지배 국가 수 |
| 동맹 | 동맹 총 전력 |

## 9. 안티봇 & 인간성 보장 설계

### 9.1 시간 기반 제약 (에이전트 무력화)
- 공격 쿨다운 12시간: 자동화해도 의미 없음
- 경매 최소 입찰 간격 30초: 빠른 입찰 무의미
- 일일 행동 제한: 공격 3회/일, 경매 입찰 20회/일
- 군대 훈련: 실시간 대기 (즉시 완료 불가)

### 9.2 인간 검증 (자연스러운)
- 전투 배치: 드래그 앤 드롭 유닛 배치 (자동화 어려움)
- 경매: 최종 확인 팝업 (의도적 지연)
- 거래: 양측 확인 필요

### 9.3 방어자 우위 원칙
