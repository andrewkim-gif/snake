# $AWW 토큰 경제 시스템 — 심층 분석 보고서

**Date**: 2026-03-10
**Analyst**: Claude Opus 4.6
**Scope**: 토큰 게임 개입 메커니즘, 운영자 경제 핸들링, 사용자 수익 경로 — 다관점 분석

---

## 1. Executive Summary

AI World War의 $AWW 토큰은 단순한 보상 토큰이 아닌 **게임 메커니즘의 핵심 자원**입니다.
5개의 Sink(소각) 메커니즘과 GDP 연동 바이백 시스템을 통해 **강한 디플레이션 구조**를 형성하며,
시즌당 발행(48K $AWW) 대비 소각(1.58M $AWW)이 **33배** 초과하는 설계입니다.

사용자는 전투(Domination), 외교(Hegemony), 전쟁(War Victory), 주권(Sovereignty) 4가지 경로로 토큰을 획득하고,
거버넌스 투표, 전쟁 선포, GDP 부스트, 시즌 스테이킹, 주권 경매 5가지 전략적 소비로 게임 내 우위를 확보합니다.

**핵심 결론**: 이 게임은 "Play-to-Earn"이 아닌 **"Play-and-Earn"** 모델입니다.
토큰 수익은 게임 플레이의 부산물이지 목적이 아니며, 전략적 소비가 게임 승리에 직접적 영향을 줍니다.

## 2. 토큰이 게임에 개입하는 5가지 경로

### 경로 A: 전투 보상 시스템 (TokenRewardManager)

```
전투 → NationScore 산출 → CalcDominationRewards → 큐잉 → 30초 분배
```

**코드 위치**: `server/internal/game/token_rewards.go:118-181`

| 파라미터 | 값 | 게임 영향 |
|----------|-----|----------|
| 기본 보상율 | NationScore × 0.01 | 전투 성적에 비례 |
| 최소 지급 | 1.0 토큰 | 참여 보장 |
| 최대 지급 | 1,000 토큰 | 인플레이션 방지 캡 |
| 주권 보너스 | +20% | 주권 유지 인센티브 |
| 헤게모니 보너스 | +50% | 장기 지배 보상 |
| MVP 보너스 | ×1.5 | 최고 기여자 우대 |
| 일일 캡 | 5,000 토큰/인 | 봇/매크로 방지 |
| 큐 상한 | 1,000 이벤트 | 서버 보호 |

**게임 개입 메커니즘**: 전투에서 높은 NationScore를 올리는 것이 직접적 토큰 보상.
승리 국가의 플레이어만 보상을 받으므로 **국가 선택 전략**이 중요합니다.

### 경로 B: 경제 시스템 연동 (BuybackEngine)

```
국가 GDP 생산 → 5% 세수 → BuybackEngine → $AWW 시뮬 매수 → 가격 상승 압력
```

**코드 위치**: `server/internal/blockchain/buyback.go:100-118`

토큰이 게임 경제에 개입하는 가장 **간접적이지만 강력한** 경로입니다.
195개국의 GDP가 매 틱마다 5% 세수를 발생시키고, 이것이 $AWW 바이백으로 전환됩니다.

| GDP 행위 | 세수 (5%) | 바이백 영향 |
|----------|-----------|------------|
| 자원 생산 (매 틱) | GDP × 0.05 | 자동 매수 압력 |
| 기술 투자 | GDP 성장 → 세수 증가 | 장기 토큰 가치 |
| 무역 개방 | GDP 부스트 | 국제 협력 보상 |

**핵심**: 플레이어가 직접 토큰을 사지 않아도, **게임을 잘 할수록 토큰 가격이 오릅니다**.

### 경로 C: 외교/전쟁 비용 (War AWW Cost)

```
전쟁 선포 → $AWW 차감 (500/1000/2000) → 전쟁 개시 → 자원 재분배
```

**코드 위치**: `server/internal/meta/war.go:186-202`

| 전쟁 유형 | AWW 비용 | 자원 비용 | 전략적 의미 |
|----------|---------|----------|------------|
| 소규모 (1v1) | 500 | 300 Inf + 500 Oil | 빠른 영토 탈취 |
| 경제 전쟁 (무역봉쇄) | 1,000 | 300 Inf + 500 Oil | 적 GDP 약화 |
| 대규모 (연합) | 2,000 | 300 Inf + 500 Oil | 결정적 전투 |

토큰이 **전쟁의 문턱**을 만들어 무분별한 공격을 억제합니다.
자본력이 충분한 팩션만 대규모 전쟁을 선포할 수 있으므로, 외교와 동맹이 중요해집니다.

### 경로 D: 거버넌스 가중 투표

```
$AWW 투입 → sqrt(토큰)만큼 가중치 → 10% 소각 → 정책 결정
```

**코드 위치**: `server/cmd/server/router.go:1286-1342`

1,000 $AWW를 투표에 사용하면 가중치 31.6을 얻지만 100 $AWW(10%)는 영구 소각됩니다.
이는 **거버넌스 참여에 실질적 비용**을 부과하여 스팸 투표를 방지하고,
진정으로 중요한 안건에만 토큰을 투입하도록 유도합니다.

### 경로 E: 전략적 경제 부스트 (GDP Boost)

```
$AWW 투입 → 50% 소각 + 50% treasury → 1~25% GDP 부스트 (4시간)
```

**코드 위치**: `server/internal/meta/economy.go:1073-1130`

| 투입량 | GDP 부스트 | 소각 | Treasury 락 | 쿨다운 |
|--------|-----------|------|------------|--------|
| 100 AWW | +5% | 50 | 50 | 4시간 |
| 500 AWW | +15% | 250 | 250 | 4시간 |
| 1,000 AWW | +25% | 500 | 500 | 4시간 |

**전략 시나리오**: 대규모 전쟁 직전에 핵심 국가에 1,000 AWW를 투입하여 GDP +25% →
방어력 강화 + 자원 생산 증가. "전쟁 전 경제 투자" 전략의 핵심입니다.

## 3. 운영자의 경제 핸들링 레버

운영자(당신)가 토큰 경제를 조절할 수 있는 **7가지 레버**입니다.

### 레버 1: 보상 상수 조절 (Emission Control)

```go
// token_rewards.go — 이 상수들을 변경하면 발행량이 즉시 바뀜
DominationTokenBaseRate = 0.01  // ↑ 올리면 보상 증가
DominationTokenMaxPayout = 1000.0  // ↓ 내리면 고래 보상 제한
DailyPlayerRewardCap = 5000.0  // 일일 개인 상한
HegemonyWeeklyAWWBonus = 100.0  // AWW 직접 발행량
```

**조절 방법**: 서버 재배포 없이 상수 변경 → 재시작으로 즉시 반영
**영향 범위**: 전체 플레이어의 토큰 수입

### 레버 2: Sink 비용 조절 (Deflation Control)

```go
// war.go — 전쟁 비용
AWWWarCostSmall = 500.0     // 소규모 전쟁 비용
AWWWarCostEconomic = 1000.0 // 경제 전쟁 비용
AWWWarCostLarge = 2000.0    // 대규모 전쟁 비용

// economy.go — GDP 부스트 티어
GDPBoostTiers = [{100, 0.05}, {500, 0.15}, {1000, 0.25}]

// auction.go — 최소 입찰
MinBidByTier = {"S": 10000, "A": 5000, "B": 2000, "C": 500, "D": 200}
```

**전략**: 인플레이션 시 → Sink 비용 낮춰서 소각 촉진
디플레이션 과다 시 → Sink 비용 올려서 토큰 보존

### 레버 3: 바이백 세율 (GDP Tax Rate)

```go
// buyback.go
BuybackTaxRate = 0.05  // GDP의 5% → 바이백
```

이 한 줄이 전체 경제의 **기본 매수 압력**을 결정합니다.
5% → 10%로 올리면 바이백 압력이 2배 증가하여 토큰 가격 상승 유도.

### 레버 4: 스테이킹 티어 & 패널티

```go
// token_rewards.go
StakingTiers = [
  {500,   1.25},  // 최소 스테이킹 → 1.25배
  {2000,  1.50},
  {5000,  2.00},
  {10000, 2.50},  // 최대 → 2.5배
]
EarlyWithdrawPenalty = 0.20  // 조기 출금 시 20% 소각
```

**조절 시나리오**:
- 토큰 가격 하락 시: 스테이킹 배수 올려서 락업 유도 (유통량 감소)
- 신규 유저 유치 시: 최소 스테이킹 낮춰서 진입 장벽 감소

### 레버 5: 시즌 주기

시즌 길이(현재 28일)를 변경하면:
- **단축** (14일): 빠른 보상 회전 → 활동성 증가, 스테이킹 락업 기간 단축
- **연장** (42일): 장기 전략 강조, 스테이킹 효과 극대화

### 레버 6: 국가 티어 재분류

```go
// economy.go
TierMultipliers = {"S": 3.0, "A": 2.0, "B": 1.5, "C": 1.0, "D": 0.5}
```

국가 티어 재분류는 자원 생산 → GDP → 바이백 → 토큰 가격에 연쇄 영향.
S티어 국가를 늘리면 GDP 총량 증가 → 바이백 증가 → 토큰 가격 상승.

### 레버 7: 방어 오라클 (Defense Oracle)

ForgePriceService를 통해 토큰 가격이 국가 방어력에 직접 영향:
- 토큰 가격 ↑ → 방어력 ↑ → 정복 난이도 ↑ → 전쟁 비용 ↑ → 더 많은 AWW 소각
- 토큰 가격 ↓ → 방어력 ↓ → 정복 용이 → 영토 변경 빈번 → 게임 역동성

이것은 **자동 밸런싱 메커니즘**으로, 토큰 가격과 게임 난이도가 자동 연동됩니다.

## 4. 사용자 수익 모델 — "어떻게 돈을 버는가"

### 수익 경로 1: 전투 머신 (Domination Farmer)

```
플레이어 행동: 매일 2시간 전투 → 높은 NationScore 유지
수익 원천: 국가 토큰 (Domination Rewards)
```

| 플레이어 등급 | 일일 NationScore | 일일 보상 | 월 보상 |
|-------------|----------------|----------|--------|
| 캐주얼 (하위 50%) | 500 | ~50 국가토큰 | ~1,400 |
| 코어 (상위 20%) | 2,000 | ~200 국가토큰 | ~5,600 |
| 하드코어 (상위 5%) | 5,000 | ~500 국가토큰 | ~14,000 |
| 프로 (상위 1%) | 10,000 | ~1,000 국가토큰 (캡) | ~28,000 |

**수익 극대화 전략**:
1. S/A티어 국가에 소속 (티어 배수 높음)
2. 주권 국가에서 플레이 (+20% 보너스)
3. 헤게모니 달성 국가 (+50% 보너스)
4. 시즌 스테이킹으로 배수 극대화 (최대 2.5배)

**프로 플레이어 최적 수익**: 1,000 × 2.5배(스테이킹) = **일 2,500 국가토큰**

### 수익 경로 2: 외교관/전략가 (Hegemony Builder)

```
플레이어 행동: 팩션 리더로 7일 연속 주권 유지 → Hegemony 달성
수익 원천: $AWW 마스터 토큰 (주간 보너스)
```

| 활동 | AWW 보상 | 빈도 |
|------|---------|------|
| Hegemony 주간 보너스 | 100 + (팩션원 × 10) | 주 1회 |
| 50명 팩션 × 3국 Hegemony | (100 + 500) × 3 = 1,800 AWW | 주 1회 |

팩션 리더가 50명 규모의 팩션에서 3국 Hegemony를 유지하면 **주 1,800 AWW**.
이것은 **조직력**이 곧 수익인 모델입니다.

### 수익 경로 3: 거래자 (Token Trader)

```
플레이어 행동: 국가 토큰 간 차익 거래
수익 원천: 195개 국가 토큰의 가격 차이
```

국가 토큰은 국가별 GDP, 전쟁 상황, 주권 변경에 따라 가격이 변동합니다.

| 거래 시나리오 | 예상 수익 |
|-------------|----------|
| 전쟁 전 공격 대상국 토큰 매도 (가격 하락 예상) | 10-30% |
| 비주권 국가 토큰 매수 → 경매 후 주권 확정 시 매도 | 20-50% |
| Hegemony 직전 해당국 토큰 매수 → 보너스 발행 후 매도 | 5-15% |

### 수익 경로 4: 주권자 (Sovereign Holder)

```
플레이어 행동: 국가 주권 유지 (domination 또는 경매)
수익 원천: 일일 Sovereignty 보너스 + 자원 배분 (30%)
```

| 주권 활동 | 일일 수익 |
|----------|----------|
| Sovereignty 일일 보너스 | 5 국가토큰/국/일 |
| 자원 배분 (SovereignShare 30%) | GDP × 30% 분배 |
| S티어 국가 주권자 | 5 + GDP 자원 (매우 높음) |

S티어 국가(USA, CHN 등)의 주권자는 게임 내 **최고 수익자** 포지션입니다.

### 수익 경로 5: 시즌 스테이커 (Passive Investor)

```
플레이어 행동: 시즌 초에 $AWW 스테이킹 → 시즌 종료 후 보상 배수 수령
수익 원천: 스테이킹 배수로 증가된 모든 보상
```

| 스테이킹 | 배수 | 추가 수익 (월 5,000 기본 보상 기준) |
|---------|------|--------------------------------|
| 0 AWW | 1.0x | 5,000 (기본) |
| 500 AWW | 1.25x | 6,250 (+1,250) |
| 2,000 AWW | 1.5x | 7,500 (+2,500) |
| 5,000 AWW | 2.0x | 10,000 (+5,000) |
| 10,000 AWW | 2.5x | 12,500 (+7,500) |

**ROI 분석**: 10,000 AWW 스테이킹 → 월 +7,500 추가 보상 → **월 75% ROI**
단, 시즌 종료 전 출금 시 20% 소각 패널티 (2,000 AWW 손실).

| 측면 | Axie Infinity | StepN | AI World War ($AWW) |

| **모델** | Play-to-Earn | Move-to-Earn | Play-and-Earn |

| **Sink 수** | 2 (breeding + marketplace fee) | 3 (repair + upgrade + mint) | **5** (투표+전쟁+부스트+스테이킹+경매) |
| **자동 바이백** | ❌ | ❌ | ✅ (GDP 세수 5%) |

### AI World War의 차별점

1. **무료 진입, 유료 가속**: NFT 구매 없이 게임 시작 가능. 토큰은 "보너스"
2. **5중 Sink**: 역대 P2E 중 가장 많은 토큰 소비처 (대부분 1-2개)
3. **GDP 연동 자동 바이백**: 게임 경제가 성장하면 토큰 매수 압력 자동 증가
4. **195개 국가 토큰**: 지리적 분산으로 단일 토큰 붕괴 리스크 분산
5. **Defense Oracle**: 토큰 가격 ↔ 게임 난이도 자동 피드백 루프
