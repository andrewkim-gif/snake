# $AWW 토큰 설계 검증 — 쉬운 한국어 요약

**날짜**: 2026-03-10
**한 줄 요약**: 우리 게임의 토큰 경제 설계를 실제 망한 게임들과 비교해봤더니, 꽤 잘 만들었습니다.

---

## 이 보고서가 뭔가요?

블록체인 게임(P2E) 중에 토큰 경제가 망해서 폭락한 게임들이 많습니다.
대표적으로 Axie Infinity(99.9% 폭락), StepN(97% 폭락) 등이 있습니다.

**"우리 $AWW 토큰은 그런 실수를 안 했는가?"** 를 30개 이상의 외부 자료로 검증한 보고서입니다.

---

## 1. 망한 게임들은 왜 망했나?

### Axie Infinity (엑시 인피니티) — 99.9% 폭락

필리핀에서 직장 대신 게임하는 사람이 생길 정도로 인기였지만 결국 폭락했습니다.

**망한 이유 3가지:**
1. **토큰을 무한으로 찍어냄** — 하루에 1.3억 개 이상 SLP 토큰이 생성됨. 한도가 없었음
2. **토큰 쓸 곳이 1개** — "브리딩"(교배)에만 쓸 수 있었는데, 대부분 사람은 토큰 벌자마자 팔아버림
3. **새 유저가 들어와야만 경제가 돌아감** — 신규 가입이 줄자 전체 경제가 무너짐

> 쉽게 말하면: 돈은 계속 찍는데, 쓸 곳이 없으니 가치가 바닥으로 떨어진 것

### StepN (스텝앤) — 97% 폭락

걸으면 돈 버는 앱이었는데 역시 폭락했습니다.

**망한 이유:**
- 토큰으로 운동화를 업그레이드하면 → 더 많이 벌 수 있음 → 더 많이 업그레이드 → 토큰 폭발적 증가
- 즉, **쓰면 쓸수록 더 많이 벌리는 악순환** 구조

> 쉽게 말하면: 돈을 쓰면 더 많은 돈을 찍어내는 기계가 되어버린 것

## 2. 망한 게임들의 공통점 5가지 vs 우리 $AWW

| # | 망한 게임들의 실수 | 우리 $AWW는? | 결과 |
|---|-----------------|------------|------|
| 1 | 토큰을 무한으로 찍음 | 총 10억 개 고정. 더 이상 안 만듦 | ✅ 안전 |
| 2 | 토큰 쓸 곳이 1~2개뿐 | 쓸 곳이 5개 (전쟁, 투표, 부스트, 스테이킹, 경매) | ✅ 안전 |
| 3 | 새 유저가 와야만 경제 유지 | 게임 내 GDP에서 자동으로 수요 발생 | ✅ 안전 |
| 4 | 하루 벌 수 있는 한도 없음 (봇이 무한 채굴) | 하루 5,000개 한도 | ✅ 안전 |
| 5 | 토큰이 게임과 관계없음 (벌고 팔기만) | 전쟁하려면 토큰 필요, 투표하려면 토큰 필요 | ✅ 안전 |

**결론: 5가지 실수 중 5가지 다 피했습니다**

## 3. "Sink(싱크)"가 뭔데 중요한가요?

**Sink = 토큰이 사라지는 장치**

세면대의 배수구를 생각하면 됩니다:
- **수도꼭지(Emission)** = 토큰이 새로 생기는 곳 (보상으로 지급)
- **배수구(Sink)** = 토큰이 사라지는 곳 (소각, 잠금)

**물(토큰)이 수도꼭지에서만 나오고 배수구가 막혀있으면?** → 넘침 → 가치 폭락
**배수구가 잘 뚫려있으면?** → 적절한 수위 유지 → 가치 유지

### 다른 게임 vs $AWW 비교

| 게임 | 배수구(Sink) 개수 | 결과 |
|------|-----------------|------|
| Axie Infinity | 1개 (교배) | 폭락 💀 |
| StepN | 2개 (수리, 민팅) | 폭락 💀 |
| Gods Unchained | 3개 | 생존 ✅ |
| Shrapnel | 3개 | 생존 ✅ |
| **$AWW** | **5개** | **업계 최다** ✅ |

### $AWW의 5개 배수구(Sink)를 쉽게 설명하면

| # | 이름 | 쉽게 설명 | 왜 자발적으로 쓰나? |
|---|------|---------|-----------------|
| ① | 투표 소각 | 투표할 때 넣은 토큰의 10%가 영구히 사라짐 | 게임 정책에 영향력을 갖고 싶으니까 |
| ② | 전쟁 비용 | 전쟁 선포하면 500~2,000 토큰이 사라짐 | 적국 영토와 자원을 빼앗고 싶으니까 |
| ③ | GDP 부스트 | 내 국가 경제력을 높이면 토큰의 50%가 소각됨 | 국가 순위를 올리고 싶으니까 |
| ④ | 스테이킹 | 예치했다가 너무 빨리 빼면 20% 깎임 | 추가 보상을 받고 싶으니까 |
| ⑤ | 주권 경매 | 국가 주권을 살 때 입찰금의 80%가 소각됨 | 국가를 지배하는 유일한 방법이니까 |

**핵심**: 5개 중 4개가 "게임에서 이기려면 써야 하는 것"이라 자연스럽게 토큰이 소비됩니다.
억지로 태우는 게 아니라 **"쓰고 싶어서 쓰는 구조"**입니다.

## 4. 소각:발행 비율 — 얼마나 좋은가?

**한 시즌(4주) 동안:**
- 새로 만들어지는 토큰: ~48,000개
- 소각되는 토큰: ~1,580,000개 (이론상 최대치)
- **비율: 33:1** (소각이 발행보다 33배 많음)

현실적으로 모든 유저가 모든 Sink를 최대로 쓰진 않으니, **실제로는 5:1~10:1** 정도일 것입니다.

### 그래도 다른 게임보다 압도적입니다

```
Axie Infinity:   0.3 : 1  (발행이 소각보다 3배 많음)  → 폭락 💀
StepN:           0.5 : 1  (발행이 소각보다 2배 많음)  → 폭락 💀
이더리움:          1.2 : 1  (소각이 약간 많음)         → 성공 ✅
$AWW (보수적):    5   : 1  (소각이 5배 많음)          → 매우 양호 ✅✅
$AWW (낙관적):   10   : 1  (소각이 10배 많음)         → 업계 최상위 ✅✅✅
```

## 5. GDP 바이백 — 우리만의 독특한 장치

대부분의 블록체인 게임은 **"새 사람이 들어와서 돈을 써야"** 경제가 돌아갑니다.
새 사람이 안 오면 → 경제 붕괴 → 이것이 Axie가 망한 핵심 원인

**$AWW는 다릅니다:**

```
일반 게임:  새 유저 유입 → 토큰 구매 → 경제 유지 (유입 멈추면 끝)
$AWW:      게임 플레이 → 국가 GDP 생산 → GDP의 5%가 자동으로 토큰 매수 압력
```

즉, **기존 유저들이 게임만 해도** 토큰에 대한 수요가 자동으로 발생합니다.
새 사람이 안 와도 경제가 바로 무너지지 않는 안전장치입니다.

**단, 주의사항:**
- 지금은 "시뮬레이션"으로 매수를 흉내내는 것이라, 실제 거래소 매수는 아닙니다
- Phase 3에서 진짜 DEX(거래소)와 연결해야 완전해집니다

## 6. 업계 트렌드와 맞나?

블록체인 게임 업계는 크게 3세대로 진화했습니다:

```
1세대 (2021): "게임하면 돈 번다" (Play-to-Earn)     → 대부분 망함
2세대 (2023): "게임이 재밌고, 돈은 보너스" (Play-and-Earn) → 생존 모델
3세대 (2025): "디지털 자산 소유가 핵심" (Play-to-Own)     → 현재 트렌드
```

**$AWW는 2세대(Play-and-Earn)에 해당하며**, 3세대 요소(주권 경매 = 자산 소유)도 포함합니다.
현재 시장 트렌드에 잘 맞는 포지셔닝입니다.

## 7. 그래서 위험한 건 뭔가요?

### 잘하고 있는 것 ✅
| 항목 | 상태 |
|------|------|
| 토큰 무한 발행 방지 | ✅ 총 10억 고정 |
| 토큰 쓸 곳 다양 | ✅ 5개 Sink |
| 봇 방지 | ✅ 일일 한도 + 인증 |
| 게임과 토큰 연결 | ✅ 전쟁/투표에 필수 |
| 신규 유저 비의존 | ✅ GDP 바이백 |

### 아직 부족한 것 ⚠️
| 항목 | 문제 | 해결 방법 |
|------|------|----------|
| **유동성 관리** | 거래소에서 사고팔 때 충분한 물량이 있어야 함 → 아직 미구현 | Phase 3에서 LP(유동성 풀) 구축 필요 |
| **적응형 발행** | 토큰이 너무 많이 소각되면 유통량 부족할 수 있음 | 자동으로 발행량 조절하는 시스템 필요 |
| **시뮬 → 실제 전환** | GDP 바이백이 아직 시뮬레이션 수준 | Phase 3에서 진짜 온체인 바이백으로 업그레이드 |
| **스테이킹 게임 연동** | 스테이킹이 게임과 직접 관련이 약함 | 스테이킹 보상을 게임 내 버프로 연결 |
