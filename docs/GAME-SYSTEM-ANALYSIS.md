## 1. 게임 개요

### 1.1 한 줄 요약

> **"AI World War"** — AI 에이전트가 실제 세계지도 위 195개국을 무대로 전투하고, 국가 경제를 운영하며, 블록체인 토큰으로 뒷받침되는 지정학적 전략 서바이벌 게임

### 1.2 게임 아이덴티티

- **장르**: 멀티플레이어 자동전투 서바이벌 로그라이크 + 지정학 전략 시뮬레이션 + AI 에이전트 플랫폼
- **플랫폼**: 웹 브라우저 (Next.js 15 + R3F)
- **서버**: Go 1.24 (WebSocket 20Hz 게임루프)
- **블록체인**: CROSS Mainnet (EVM 호환, 저가스비)
- **데이터**: PostgreSQL (영속) + Redis (실시간)

### 1.3 버전 진화

| 버전 | 핵심 변화 |
|------|----------|
| **v10** | Room 기반 5분 아레나 전투, 자동전투 + Tome/Ability 빌드 |
| **v11** | 195개국 세계지도, 팩션/외교/경제, 블록체인 토큰 |
| **v14** | 10분 에포크(Peace→War), 무기 시스템 오버홀, 문명 정책 |
| **v15** | 외부 AI 에이전트 API, ELO 레이팅, 관전/베팅, OpenClaw 연동 |

### 1.4 "Agent"의 삼중 의미

1. **게임 캐릭터**: MC 스타일 전투 유닛 — 아레나에서 싸우는 엔티티
2. **AI 에이전트**: Claude/GPT/커스텀 LLM — API로 전략을 실행하는 자율 플레이어
3. **국가 대리인**: 유저의 의지를 대행하며 국가를 정복·통치하는 존재

### 1.5 유저 역할 (4가지 플레이 스타일)

| 역할 | 설명 | 핵심 활동 |
|------|------|----------|
| **Strategist** | 에이전트를 훈련·배치하는 코치 | 빌드 설정, 국가 선택, 전쟁 타이밍 |
| **Observer** | 세계를 탐험·관전하는 유저 | 맵 탐험, 라이브 관전, 통계 분석 |
| **Warrior** | 직접 전투에 참여하는 플레이어 | Commander Mode로 실시간 조종 |
| **Governor** | 국가 정치/경제를 운영하는 총독 | 세율 설정, 무역 정책, 외교 협상 |

## 2. 핵심 게임 루프

게임은 **3개 레이어**가 상호 영향을 주고받는 구조:

```
Layer 3: META GAME (세계 전략)
  → 195국 세계지도, 팩션 경쟁, 외교/동맹, 시즌 랭킹, 영토 지배, 경제 운영

Layer 2: BATTLE (국가 아레나 전투)
  → 10분 에포크(Peace 5분 + War 5분), 자동전투, 무기/패시브/시너지 빌드, 리스폰

Layer 1: AGENT (에이전트 관리)
  → AI 에이전트 훈련, 빌드 프로필, 전략 설정, API 연동, Commander Mode
```

### 2.1 시간 루프

| 주기 | 이벤트 | 설명 |
|------|--------|------|
| **10분** | 에포크 사이클 | Peace(5분) → War Countdown(3초) → War(3분) → Shrink(2분) → 결산 |
| **1시간** | 도미네이션 평가 | 6 에포크 결과 합산 → 주권 결정 |
| **24시간** | 외교 틱 | 조약 갱신, 제재 효과, 인구 변동, 주권 확정 |
| **1주** | 에라 전환 | Discovery → Expansion → Empires → Reckoning |
| **1개월** | 시즌 리셋 | 세계 초기화, 명예의 전당, 새 시즌 테마 |

### 2.2 v14 에포크 상태 머신

```
PEACE(5m) → WAR_COUNTDOWN(3s) → WAR(3m) → SHRINK(2m) → EPOCH_END(5s) → TRANSITION(5s) → PEACE
```

- **Peace Phase**: PvP OFF, 오브 파밍, NPC 스폰, 무기 수집, 레벨업
- **War Phase**: PvP ON, 킬 보상 활성, 영역 점령 포인트 쟁탈
- **Shrink Phase**: PvP ON + 아레나 수축 (3000→1000px), 최후의 결전
- **Epoch End**: 스코어보드, MVP, 국가 점수 집계

### 2.3 도미네이션 계층 (v14)

```
에포크 (10분) → 세션 (1시간, 6에포크) → 주권 (24시간) → 패권 (1주)
```

| 계층 | 주기 | 결정 방식 |
|------|------|----------|
| Epoch Winner | 10분 | 킬+캡처 점수 1위 팩션 |
| Session Sovereign | 1시간 | 6 에포크 우승 횟수 |
| Daily Sovereign | 24시간 | 세션 우승 누적 |
| Weekly Hegemon | 1주 | 일간 주권 누적 → 문명 정책 해금 |
